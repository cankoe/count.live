Count-Up Mode

Use mode=up to count up from a past date (e.g., "Days since I quit smoking"). The timer runs indefinitely, showing elapsed time since the target date.

Recurring Countdowns

Use recur to auto-reset the countdown. When the countdown reaches zero, it automatically calculates the next occurrence and continues.

Multiple Countdowns

Use multi=1 with numbered parameters: date1, title1, date2, title2, etc. Up to 5 countdowns supported.

Progress Tracking

Use progress=1 and/or percent=1 with a start date to show a visual progress bar and percentage completion.

Embedding

Use the Embed button to get iframe code for your website. Add embed=1 to the URL for minimal UI without the promo badge.

Date Formats

The following date formats are supported (interpreted as UTC):

  • 2025-12-31T23:59:59 - Full date and time
  • 2025-12-31T23:59 - Date and time without seconds
  • 2025-12-31 - Date only (midnight)

How Units Work

The largest selected unit shows the total value. Smaller units show the remainder after subtracting larger units.

Example: If you select weeks and minutes for a 15-day countdown, you'll see "2 Weeks, 1440 Minutes" (not "2 Weeks, 30 Minutes").

Clickable Links

URLs in the title, subtitle, and end message are automatically converted to clickable links. Just paste a full URL (starting with http:// or https://) and it will be linked.

When Does the Countdown End?

The countdown ends at the exact target time. Hidden smaller units are accounted for using ceiling logic.

Example: If you select only minutes for a countdown to 12:30:45, you'll see "1 minute" until 12:30:45, then "0 minutes" briefly, then the end message.

This ensures the countdown reaches zero only when the target time has truly arrived.

Precise Time Calculations

Years and months are calculated using actual calendar dates:

  • Years - Exact calendar years (accounts for leap years)
  • Months - Exact calendar months (accounts for varying month lengths)
  • Weeks/Days/Hours/etc. - Calculated from remaining time after years/months

Example: From Jan 15 to Mar 15 is exactly 2 months, regardless of whether February has 28 or 29 days.
